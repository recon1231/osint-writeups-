# OSINT Resources

Tools and references used regularly for OSINT research and investigations.

## Search & Correlation
- Google/Bing advanced search operators (site:, intitle:, filetype:, etc.)
- Wayback Machine (web.archive.org) — historical page snapshots
- crt.sh — certificate transparency search, useful for subdomain/infrastructure discovery

## Username & Identity Correlation
- WhatsMyName / Sherlock — username enumeration across platforms
- Reverse image search (multiple engines — no single tool covers everything)

## Domain & Infrastructure
- WHOIS/RDAP lookup tools
- Shodan / Censys — internet-connected device and service discovery
- DNSDumpster — DNS reconnaissance

## Social Media & Public Records
- Platform-native advanced search (most platforms have underused search operators)
- Public breach/leak validation services (for checking exposure, not accessing stolen data)

## Visualization & Link Analysis
- Maltego — link analysis and relationship mapping
- Manual timeline/graph building for smaller investigations

## Practice & Learning
- TraceLabs OSINT CTFs — legal, structured practice environment for missing-persons-style OSINT (search-and-rescue focused)
- OSINT Dojo challenges
