# Practice Scenario: Username Correlation Across Platforms

**Date:** 2026-07-18
**Category:** OSINT / Practice Scenario
**Status:** Fictional — Training Exercise

⚠️ **This is a fictional practice scenario created for methodology demonstration purposes only.** "quietfalcon22" is a fictional placeholder handle, not a reference to any real person's account.

## Summary
Demonstrates the process of correlating a single username across multiple platforms to assess whether they belong to the same individual — a core OSINT skill for both investigative and threat-actor-attribution work.

## Objective
Starting from one handle, determine (using only passive research) whether other accounts using the same or similar handle likely belong to the same person, and to what confidence level.

## Methodology
1. **Direct handle search** across major platforms (social media, forums, code hosting, gaming platforms) to catalog every account using the exact handle
2. **Handle variation search** — checking common variations (underscores replaced with dots, added numbers, stylized characters) since people rarely use only one exact form of a handle
3. **Cross-referencing supporting signals** — profile photos (via reverse image search), bio text/writing style, posting times and timezone patterns, and topics of interest
4. **Confidence scoring, not binary conclusions** — treating each match as a data point that raises or lowers confidence, rather than declaring identity confirmed from a single handle match alone
5. **Independent corroboration requirement** — never concluding a match based on the handle alone; requiring at least one additional independent signal (photo, writing style, or content overlap)

## Tools Used
- Username enumeration tools (checking handle availability/usage across platforms)
- Reverse image search
- Manual cross-platform bio/content comparison

## Findings (Illustrative — Fictional Data)
In this fictional walkthrough, the handle appears on three platforms with matching bio language, and a stylized variation of the same handle appears on a fourth platform with a reused profile photo — raising confidence the accounts belong to one individual, though not proof on its own without further corroboration.

## Key Takeaways
- A single handle match is weak evidence on its own — reused handles are common but not unique enough alone
- Writing style and bio phrasing are underused correlation signals compared to photos
- Confidence scoring (not binary yes/no conclusions) is the correct mental model for this kind of correlation work, since false positives in attribution work carry real consequences
